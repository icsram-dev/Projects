Grade Book alkalmazás

Ez egy egyszerű Grade Book (jegyellenőrző) alkalmazás, amelyet React és Firebase segítségével építettem. Az alkalmazás lehetővé teszi a felhasználók számára, hogy kezeljék a diákok jegyeit: hozzáadják, frissítik és törlik a jegyeket különböző tantárgyakban. Az adatokat a Firebase Firestore-ban tárolja. 

Funkciók:

- Diákok jegyeinek hozzáadása: A felhasználók választhatják a diákot, tantárgyat és jegyet, hogy hozzáadják az adatbázishoz.

- Jegyek szerkesztése: A diákok jegyei módosíthatók adott tantárgyak szerint.

- Jegyek törlése: A diákok jegyei törölhetők a rendszerből.

- Jegyek megjelenítése: Az összes diák jegye megjeleníthető tantárgyak szerint.

- Átlagos jegyek kiszámítása: Minden diák átlagos jegyei tantárgyak szerint.

Frontend: React, Tailwind CSS
Backend: Firebase Firestore
Állapotkezelés: React useState és useEffect
